Windows 11 查看死机原因
1
Windows 11 查看死机原因
方法1：事件查看器（最直接） 按 Win + X → 选择「事件查看器」 左侧展开：Windows日志 → 系统 右侧点击「筛选当前日志」 事件级别勾选：严重、错误 找死机时间点附近的红色错误记录 方法2：蓝屏转储文件（如果是蓝屏死机） 按 Win + R 输入： %SystemRoot%\Minidump 里面有 .dmp 文件，文件名包含日期 可以用免费工具 WhoCrashed 打开分析，会直接告诉你是哪个驱动或程序导致的 方法3：可靠性监视器（最简单直观） 搜索栏输入「可靠性监视器」打开 会显示一个时间轴，红色X就是崩溃点 点击对应日期可以看详细描述 方法4：CMD快速查看 管理员运行CMD，输入： wevtutil qe System /q:"*[System[Level=1 or Level=2]]" /f:text /rd:true /c:10 显示最近10条严重错误。
